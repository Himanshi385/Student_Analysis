# 📊 Student Performance Data Analysis

This project is part of a **Data Science Internship (Task 1)**.  
It analyzes student academic performance using Python through data cleaning, analysis, and visualization.

## 🔹 Dataset
- Student Performance Dataset (`student-mat.csv`)
- Source: UCI Machine Learning Repository

## 🔹 Tools Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Jupyter Notebook

## 🔹 Analysis Includes
- Average final grade (G3)
- Students scoring above 15
- Study time vs performance correlation
- Gender-wise performance comparison
- Visualizations (histogram, scatter plot, bar chart)

## 🔹 How to Run
Open the notebook in Jupyter Notebook or Google Colab and run all cells.
